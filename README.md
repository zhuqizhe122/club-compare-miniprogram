# club-compare-miniprogram

本地优先微信小程序：**全部社团资料** + **问卷推荐（≤4 对照 + 提问建议）**。

## 流程

1. 第一部分：`index` → `library` → `club`  
2. 第二部分：`index` → `quiz`（12 题）→ `recommend`（勾选≤4）→ `compare` / `club` → `result`（提问）

## 打开

微信开发者工具导入本仓库根目录。

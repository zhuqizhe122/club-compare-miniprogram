## C. 微信开发者工具人工验收

### C1. 八页路由与首页边界

- [ ] 首页可分别进入“开始专业测评”和“直接浏览资料”，控制台无报错。
- [ ] 运行时可到达 `index → assessment → recommend → selection → compare → result`。
- [ ] 从首页可到达 `library → club → selection`，并与推荐路径共享候选。
- [ ] 页面实际不存在可进入的第九条业务主路由；旧 `pages/list` 不出现在当前导航。
- [ ] 首页明确显示 8–14 题、本地计算、非官方评价和 D 级资料边界。

### C2. 8–14 题自适应测评

- [ ] 每一题均显示题号、至少 8/最多 14、单选/多选类型和“为什么问”。
- [ ] 完成一组最短路径后恰在第 8 题结束，没有无理由追加题。
- [ ] 完成一组持续不确定路径后逐题追加，但第 14 题后必须结束。
- [ ] 追加题理由能对应“硬条件未知”“头部候选接近”或“偏好画像空白”。
- [ ] 多选题中“没有固定冲突”与其他冲突时段互斥。
- [ ] 返回上一题并修改答案后，后续自适应题、旧 Top 6、候选、压力情境和倾向正确失效或重算。

### C3. Top 6、MCDA 与证据分离

- [ ] 推荐页显示不超过 6 个候选，并稳定展示名次和 0–100 偏好匹配分。
- [ ] 每项至少显示可读匹配理由、时间代价/活动频率和硬条件未知提示。
- [ ] 页面明确把“偏好匹配分”与“资料等级 D”分开，高分不被描述为高可信或官方推荐。
- [ ] D 级 `prototype-inferred` 的表面匹配没有被显示成“硬条件已核实通过”。
- [ ] “修改测评答案”可返回评估；修改后旧推荐不会继续作为当前结果展示。
- [ ] 无推荐状态能恢复到测评入口，不出现白屏或死路。

### C4. 资料库、详情与 2–3 候选

- [ ] 资料库空筛选显示 98 个结果，并按每次 12 个加载更多。
- [ ] 名称/关键词搜索、七类分类筛选、投入、活动形式和入门门槛筛选均可组合使用。
- [ ] 无结果时显示可恢复空态，“清除全部条件”恢复列表。
- [ ] 详情页展示时间、参与方式、门槛、成本、五维体验和 D 级来源说明。
- [ ] 从推荐、资料库和详情页加入的候选在共享托盘/候选页中保持一致。
- [ ] 只选 1 个时不能比较，并显示还需选择 1 个。
- [ ] 2 个候选可以比较；3 个候选也可以比较。
- [ ] 已选 3 个后添加第 4 个被拒绝，原 3 个及其顺序不变。
- [ ] 候选页显示来源、顺序、普通周/高峰周投入和未解决风险。

### C5. 深度比较与压力重排

- [ ] 2 个候选和 3 个候选均能进入比较页，选择顺序得到保留。
- [ ] 每个候选摘要显示基础名次、0–100 分、硬条件状态、资料等级、可能优势、代价和未知项。
- [ ] “关键 3 个权衡”展示三个可读字段，并列出各候选对应值。
- [ ] 分组比较按时间承诺、参与体验、成长门槛展示，同一字段的值与 D/U 证据状态对齐。
- [ ] 切换“考试周时间减半”后，基础分、情境分、分差和名次变化可读。
- [ ] 切换“任务临时加码”后，结果真实刷新而非只切换标签。
- [ ] 切换“连续两次无法参加”后，结果真实刷新而非只切换标签。
- [ ] 三个情境均明确标注“D 级演示估计，非预测”。
- [ ] 小屏和 3 候选情况下没有遮挡关键按钮，长内容可完整阅读。

### C6. 主倾向报告与 3–5 问

- [ ] 结果页提供每个候选的“更倾向”选项和“先都不加”。
- [ ] 未选倾向直接确认时显示“请先选择一项倾向”。
- [ ] 倾向某候选后只生成该主倾向的一张预期卡，而非给所有候选各生成一张。
- [ ] 主倾向卡包含匹配分、D 级资料、加入前预期、关键理由、代价和未知项。
- [ ] 选择“先都不加”生成暂缓加入行动卡，不暗示必须加入。
- [ ] 有测评答案时生成 3–5 个现场核实问题；问题显示“为什么问”。
- [ ] 直接浏览且无测评答案时生成 3 个现场核实问题，不伪造偏好理由。
- [ ] 报告显示四周复盘标准、证据边界和“真实加入仍在线下完成”。
- [ ] `project`、`regular`、`busy` 等内部枚举不会直接泄漏到用户文案。

### C7. 会话、返回、重启与系统边界

- [ ] 从结果返回比较、从比较返回候选页后，当前会话候选与压力情境合理保留。
- [ ] “重新开始”清空答案、题序、Top 6、候选、压力情境和倾向，并回到首页。
- [ ] 彻底关闭并重开后，界面不声称上轮结果已持久保存。
- [ ] 断网后八页主路径、资料搜索、评分、比较、压力重排和报告仍可操作。
- [ ] DevTools Network 面板没有业务网络请求。
- [ ] 页面没有登录、云同步、远程数据库、AI/LLM、站内扫码或代报名能力。

发现失败时，在 `fix-log.md` 记录环境、复现、预期、实际和风险；修复并复测前，不勾选对应人工项。

# Feature Spec — 资料库 + 问卷推荐（002）

**Created**: 2026-07-24  
**Status**: Active  
**Replaces primary flow of**: `specs/001-club-compare-flow`（旧勾选 2–3 对照）

## US1 — 部分一：全部社团资料

用户从首页进入资料库，浏览全部本地社团，点进详情看四字段；**可按分类筛选，可按关键字搜索**。

**AC**

1. 首页有「全部社团资料」入口  
2. 列表展示全部社团（名称+简介）；展示所属分类  
3. 详情含每周时间、活动频率、职责、氛围；空则「未提供」  
4. 详情含**图片区**（有图展示，无图占位）与**荣誉介绍**（有文案展示，无则占位）  
5. 可按分类筛选；可选「全部」  
6. 可输入关键字过滤名称或简介；无结果时有空态提示  

## US2 — 部分二：问卷推荐

**AC**

1. 首页有「问卷推荐」入口  
2. 问卷题数 ≥10；未答完不可提交  
3. 结果含推荐社团、推荐理由、简单介绍  
4. 可勾选不超过 **4** 个进入比较；第 5 个不可选  
5. 可从推荐/对照进入详情  
6. 对照完成后进入提问页：**先选择一个感兴趣的社团**（或先都不加）；选定后**再展示该社团信息**，并**按问卷回答生成提问建议**；文案说明不完成加群  

## FR

- FR-001 双入口  
- FR-002 全量列表+详情四维；资料库 MUST 支持分类筛选与关键字搜索；详情 MUST 含图片区与荣誉介绍（可占位）  
- FR-003 问卷≥10；规则匹配  
- FR-004 推荐展示理由+简介  
- FR-005 比较篮 0–4；≥2 可对照  
- FR-006 提问清单规则生成，且 MUST 结合 `quizAnswers` 定制问题  
- FR-007 提问页 MUST 分两步：① 先选感兴趣社团；② 选定后再展示社团信息与提问建议  
- FR-008 无登录、无 LLM、无站内加群

# 项目主计划 — club-compare-miniprogram

> 2026-07-24 按产品要求更新：双部分（资料库 + 问卷推荐）。

## 产品目标

本地优先微信小程序，含两部分：

1. **全部社团资料**：浏览本地全量社团并查看详情。  
2. **问卷推荐**：≥10 题问卷 → 规则推荐（理由+简介）→ 自选 ≤4 社比较 → 可进详情 → 给出可能需要的问题。

真实加群在线下完成。

## 核心流程

**部分一**  
`index` → `library` → `club`

**部分二**  
`index` → `quiz` → `recommend`（勾选≤4）→ `compare` 与/或 `club` → `result`（提问清单）

## Must

- 双入口  
- 全量 `data/clubs.js` 可浏览 + 四维详情  
- 问卷 ≥10 题；规则推荐含理由与 tagline  
- 比较篮最多 **4**；可点详情  
- 结束后给出摊位向提问（规则模板，非 AI）  
- 无登录 / 无直播 AI / 无站内加群  

## 非目标

登录、云同步、远程库、LLM、站内加群、新 npm 依赖。

## 数据

- 权威数据：`data/clubs.js`（见 `docs/data-contract.md`）  
- 匹配用 tags：**运行时启发式推断**，不强制写入每条 Club 记录  
- 会话：`quizAnswers`、`recommendation`、`basketIds`（≤4）、`tendency`  

## 里程碑

1. 部分一资料库可跑  
2. 问卷≥10 + 推荐列表  
3. 选≤4 对照 + 详情 + 提问清单
